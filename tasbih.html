Configuration du Mode Concentration
Pour une expérience optimale, veuillez désactiver ces distractions
Désactiver le Wi-Fi
Allez dans Paramètres → Wi-Fi → Désactiver pour éviter les notifications et distractions.
Paramètre requis
Désactiver les données mobiles
Allez dans Paramètres → Réseau mobile → Données mobiles → Désactiver.
Paramètre requis
Activer le mode silencieux
Utilisez le bouton mode silencieux de votre appareil ou allez dans Paramètres → Sons → Activer le mode silencieux.
Paramètre requis
Désactiver le Bluetooth
Allez dans Paramètres → Bluetooth → Désactiver pour éviter les distractions des appareils connectés.
Paramètre recommandé
Activer le Mode Avion
Le mode avion désactive toutes les connexions sans fil d'un seul coup, éliminant la plupart des distractions.
